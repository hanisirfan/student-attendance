How To Use?

  1. Copy the config-example.json and timetable-example.json file.

  2. Rename the file to config.json and timetable.json respectively.

  3. Fill in the details needed for each files.

  4. Make sure this page is always on to keep the bot running (uses vanilla JS).

Format

Make sure to use 24-hour format for the time.

You can add multiple classes for a day.

  1. config.json

                            {
                            "page": "http://url.com",
                            "studentName": "STUDENT NAME",
                            "studentIC": "STUDENT IDENTIFICATION NO.",
                            "studentClass": "STUDENT CLASS"
                        }

  2. timetable.json

                            {
                            "Monday": [
                                {"code": "CLASS CODE", "time": "00:00"}
                            ],
                            "Tuesday": [
                                {"code": "CLASS CODE", "time": "00:00"}
                            ],
                            "Wednesday": [
